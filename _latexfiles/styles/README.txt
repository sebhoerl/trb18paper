This directory contains one or more Bibtex .bst style files.

DO NOT CHANGE ANYTHING IN THE FOLLOWING FILES:
template_elsart-harv-eng.bst (for english papers in JOCM)
template_ivt-eng.bst (for english papers in IVT style)
template_ivt-ger.bst (for german papers in IVT style)
template_ivt-unsrt-eng.bst (for english papers in 'unsort' style - i.e. TRB)
template_ivt-unsrt-ger.bst (for german papers in 'unsort' style - i.e. TRB)

The translations.txt file now can be found in the "bibs" directory.
